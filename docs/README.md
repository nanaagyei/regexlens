# RegexLens documentation (Nextra)

This directory builds the **public documentation site** (Nextra) under `pages/`.

- **Local dev:** from `docs/`, run `npm install` then `npm run dev` (see the docs `package.json` for the dev port).
- **Published site:** [docs.regexlens.dev](https://docs.regexlens.dev) (when deployed from this repo).

Product behavior, UI flows, and feature explanations should be documented in **`pages/`** (MDX) so they stay aligned with what users see.

Repository-wide contribution guidelines live in the root **[`CONTRIBUTING.md`](../CONTRIBUTING.md)** file in the main RegexLens app repo.

- **Live app:** [regexlens.dev](https://regexlens.dev)
