# Regex Copilot

import { Callout } from 'nextra/components'

<Callout type="info">
**Bring Your Own Key:** Copilot uses `POST /api/ai/chat` and requires signing in plus your own **Anthropic API key**. Your key is stored only in your browser (24-hour expiry) and never saved on our servers. Content is processed by Anthropic under your own API key — do not send secrets or regulated data.
</Callout>

**Regex Copilot** is a streaming AI assistant embedded in the **Copilot** tab. It receives **context** from your current pattern, flags, sample test text (when present), match counts, and summarized warnings/explanation steps so answers stay grounded in what you are actually editing.

## Setup

1. Sign in to RegexLens.
2. Navigate to the **Copilot** tab.
3. Enter your Anthropic API key when prompted. Get one at [console.anthropic.com](https://console.anthropic.com/settings/keys).
4. Your key is cached in your browser for 24 hours, then automatically removed.

## Capabilities

- Ask for pattern rewrites, explanations in different words, edge-case checks, or debugging help.
- Use structured **actions** from the UI (e.g. pattern generation helpers) where implemented.
- **Apply** suggested patterns back into the editor when the UI offers an apply control.

## Guardrails

- **Rate limits** apply (`ai_chat` tier on the server).
- **Sign-in gate** — unauthenticated users see a lock state with a link to sign in when this feature requires a session.
- **API key required** — you must provide your own Anthropic API key; we do not supply one.
- Treat Copilot output as **advisory**; always validate against real data and your engine.

## Privacy

Your API key never leaves your browser except to be forwarded to Anthropic via our proxy. We do not log, store, or retain it. See our [Privacy Policy](/privacy) for details.
